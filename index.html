Projects

TVDS - Table Vacancy Detection System

An Arduino & Django project using ESP8266 microcontrollers. TVDS modules are attached under tables and assigned a number, connects to a locally hosted server via Wi-Fi. It shows real-time seat availability via web interface displayed on a screen to help customers find and choose vacant tables in restaurants.

Awards:

  • Silver Award – IEYI 2024, Taiwan
  • Outstanding Research Paper – Letran Calamba SHS

Technologies Used: Arduino, Django, Python, HTML, CSS, JavaScript, SQLite

OLLIE - A Digital Deskmate

An Arduino project using the ESP32 microcontroller. Connects to devices via bluetooth, sends user input as keystrokes. Currently under development.

Technologies Used: Arduino w/PlatformIO, C++
